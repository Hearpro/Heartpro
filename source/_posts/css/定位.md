
# absolute定位

`overflow: hideen` 影响不了 `position:absolute`定位  如果 `overflow: hideen`  还有 `transform`属性 则会影响

```css
 #app {
      width: 100px;
      height: 100px;
      background-color: red;
      overflow: hidden;
      transform: translate3d(0,0,0);
    }

    .container  {
      position: absolute;
      width: 150px;
      height: 150px;
      background-color: skyblue;
    }
```
