# TypeScript无法重新声明块范围变量

在报错的文件底部添加一行代码：export {}。

> 在 Typescript 中，只要文件存在 import 或 export 关键字，都被视为 module

```typescript
// 只要文件存在 import 或 export 关键字，都被视为 module
export {}

```

# 浏览器端exports is not defined

设置tsconfig.json的module模快不为CommonJs

```typescript
-- "module": "CommonJS"
++ "module": "es6",    
```
